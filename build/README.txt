Usage:
neural_index directory [-inf index_filename] [-t]

directory		path to dir, what to index

-inf index_filename	set serialezed index file name to index_filename

-t			if present - print duration time


Examples .

  neural_index "C:\Users\NovAtor\Desktop\ind" -ifn .nindex -t
Indexing dir - "C:\Users\NovAtor\Desktop\ind", index ser.filename - .nindex.ser, print duration
